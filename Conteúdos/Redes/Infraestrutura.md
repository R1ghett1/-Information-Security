# infraestrutura de Rede

### Conceito

O caminho que uma informação percorre desde sua origem ao destino pode ser tão simples como único cabo conectado a dois hosts ou tão complexo quanto uma rede que literalmente atravessa o globo terrestre. A infraestrutura de rede é uma plataforma que suporta a rede. Ela permanece o canal estável e confiavel sobre qual as comunicações podem ocorrer.

A infraestrutura de rede contém três categorias de componentes de rede:

- Dispostivos finais;
- Dispositivos Intermediários;
- Dispositivos físicos de rede.

Dispositivos e meios físicos são os elementos de hardware da rede. O hardware geralmente é composto pelos componentes vísiveis da plataforma de rede, tais como notebooks, computadores, swicth ou cabos utilizados para conectar os dispositivos. Ocasionalmente, alguns componentes podem não ser tão visíveis. No caso de meios físicos sem fio, as mensagens são transmitidas pelo ar com a utilização de frequência de rádio invisível ou ondas infravermelhas.

### Dispositivos Finais 

Os dispositivos de rede com os quais as pessoas são mais familiarizadas são chamados de dispositivos finais. Estes dispositivos formam a interface de interação entre os usuários e a rede de comunicação subjacente. Alguns exemplos de dispositivos finais sãoÇ

- Computadores;
- Impressoras de rede;
- Telefones e equipamentos de segurança;
- Câmeras de segurança;
- Dispositivos móveis.

Um dispositivo final ou host é a origem ou destino de uma mensagem transmitida pela rede. Quando um host inicia uma comunicação, ele usa o endereço host de destino para específicar onde a mensagem deve ser enviada.
